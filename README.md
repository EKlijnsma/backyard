# Achtertuin Ultra

Simple one-page event site for the Achtertuin Ultra backyard running event.

## What’s Included
- Event overview, format, and vibe
- Runner signup form (opens a pre-filled `mailto:` email)
- Contact form for questions (opens a pre-filled `mailto:` email)
- FAQ section

## Preview Locally
Open `index.html` in your browser.

## Feedback Workflow
- Open a GitHub issue with your feedback
- Include a short description and, if relevant, a screenshot
- Suggest exact text changes when possible

## Tech Notes
- No backend required
- Forms open the visitor’s email client with pre-filled content
